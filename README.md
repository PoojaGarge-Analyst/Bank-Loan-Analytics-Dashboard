# Bank-Loan-Analytics-Dashboard
## 1. Project Title / Headline
**Bank Loan Analytics Dashboard | Power BI**
An interactive Power BI dashboard built to monitor and analyze bank loan portfolio performance, risk, and repayment trends.

---

## 2. Short Description / Purpose
The Bank Loan Analytics Dashboard is an interactive Power BI report designed to help analyze 38,000+ loan applications. It focuses on tracking key lending metrics like funded amount, interest rate, and DTI ratio, while classifying loans as Good or Bad to support risk assessment. The tool is intended for use by bank analysts, credit risk teams, and decision-makers who need a clear, data-driven view of loan portfolio health.

---

## 3. Tech Stack
The dashboard was built using the following tools and technologies:
- 📊 **Power BI Desktop** – Main data visualization platform used for report creation.
- 📁 **Excel** – Used for data cleaning and preparation before loading into Power BI.
- 🧮 **DAX (Data Analysis Expressions)** – Used for calculated measures, KPI cards, and Good/Bad loan classification logic.
- 📄 **Data Modeling** – Relationships established between loan data tables to enable cross-filtering across visuals.
- 📁 **File Format** – .pbix for development, .png for dashboard previews.

---

## 4. Data Source
Dataset sourced from a YouTube tutorial dataset (publicly available practice dataset for loan analytics projects).
The data includes 38,000+ bank loan applications with details on funded amount, amount received, interest rate, loan purpose, grade, loan status, home ownership, DTI ratio, and state.

---

## 5. Features / Highlights

**Business Problem**
Banks handle thousands of loan applications and need a fast, reliable way to monitor portfolio performance and flag risky loans, but raw loan data is hard to interpret without a proper reporting tool.

**Goal of the Dashboard**
To give a clear, interactive view of the loan portfolio — funded amounts, repayment performance, and risk classification — so decision-makers can assess health at a glance instead of digging through spreadsheets.

**Walkthrough of Key Visuals**
- **KPI Cards (6):** Funded Amount, Amount Received, Interest Rate, DTI Ratio, and Loan Performance metrics for real-time monitoring.
- **Line Chart:** Loan trends over time (e.g., funded amount by month).
- **Donut Chart:** Loan status/purpose distribution.
- **Bar Chart:** Comparison across loan grade or purpose.
- **Treemap:** Loan volume breakdown by category.
- **Geo-map:** State-wise loan distribution across the US.
- **Slicers:** Purpose, Grade, Loan Status, Home Ownership, State — for dynamic, self-service filtering.

**Business Impact & Insights**
- **Risk Assessment:** Loans classified as Good (86%) vs. Bad (14%) using DAX logic, helping credit teams quickly flag high-risk applications.
- **Portfolio Monitoring:** Real-time KPI tracking helps management monitor funded amount vs. amount received.
- **Regional Analysis:** Geo-map lets stakeholders see which states have higher loan volume or risk concentration.
- **Decision Support:** Self-service slicers let different teams (credit, marketing, regional managers) explore the data relevant to them without needing custom reports.
